# Evolução da Inteligência Artificial

Este repositório contém uma apresentação sobre a **Evolução da Inteligência Artificial: Do Simbólico ao Generativo**, desenvolvida por **Renan Costa Alencar** para a disciplina de Inteligência Computacional da Escola Politécnica de Pernambuco - POLI/UPE.

O projeto foi desenvolvido como material de apoio para a criação de um vídeo explicativo sobre o tema.

## 📝 Sobre a Tarefa

De acordo com o projeto (`TASK.MD`):
- **Descrição**: Cada aluno deve criar um vídeo usando a própria voz de até 3 minutos sobre a evolução da inteligência artificial.

## 🚀 Tecnologias Utilizadas

A apresentação foi construída utilizando:
- **HTML5** e **CSS3** (com paleta de cores moderna, design escuro e animações pulsantes)
- **[Reveal.js](https://revealjs.com/)** para a criação de slides dinâmicos e responsivos na web
- **Google Fonts** (Inter e Space Grotesk)

## 🎨 Tópicos Abordados

1. O que é IA?
2. Linha do Tempo da IA (1950s - Hoje)
3. Big Data & Machine Learning
4. Redes Neurais e Deep Learning
5. IA Analítica vs Generativa
6. Automação & Energia (Indústria 4.0, Casos de Sucesso)
7. Sociedade & Trabalho (Human-in-the-Loop)
8. Conclusão e Referências Bibliográficas

## ⚙️ Como Executar

Para visualizar a apresentação, basta abrir o arquivo principal em qualquer navegador web moderno:

1. Clone o repositório ou baixe os arquivos.
2. Navegue até a pasta `dist`.
3. Dê um duplo clique no arquivo `index.html` ou abra-o no seu navegador preferido.
4. Utilize as setas direcionais do teclado ou a barra de espaço para navegar entre os slides.
